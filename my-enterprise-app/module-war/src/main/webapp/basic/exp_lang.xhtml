Expression Language


Aritmeticos

Suma: ${10+5}

Resta: ${10-5}

Multiplicación: ${10*5}

División: ${10/5} o también ${10 div 5}

Resto: ${10%3} o también ${10 mod 3}


Relacionales

Igual que: ${4==4}

Diferente de: ${10!=3}

Mayor que: ${10>4}

Menor que: ${2 lt 4}

Igual que: ${2 eq 4}

Menor o igual que: ${4 le 10}

Mayor o igual que: ${2 >= 4}


Lógicos

${ (6 > 2) || (4 != 0) }

${ (3 == 3) and (3 gt 8)}

${not (5 lt 1)}

${empty []}


Cadenas

${"Hola " += "Mundo"}
